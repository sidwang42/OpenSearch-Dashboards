## Banner Configuration

The plugin displays a configurable banner at the top of every page in OpenSearch Dashboards.
You can customize the banner by visiting the Test plugin page and using the banner controls.

Options include:
- Text: The message to display
- Color: Choose from primary, success, warning, or danger
- Icon: Select an icon to display
- Visibility: Toggle the banner on or off
